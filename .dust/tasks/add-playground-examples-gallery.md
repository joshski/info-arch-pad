# Add playground examples gallery

Add a set of pre-built example `.ia` files to the playground that users can load as starting points. Include common IA patterns such as e-commerce sites, documentation sites, and SaaS dashboards.

Examples should be bundled directly into the playground HTML so it works offline without a server.

## Goals

- [Easy to try](../goals/easy-to-try.md)

## Blocked By

(none)

## Definition of Done

- [ ] At least 3 example `.ia` diagrams are available in the playground
- [ ] Users can select and load an example from a dropdown or gallery UI
- [ ] Loading an example populates the editor and renders the diagram
- [ ] Examples demonstrate distinct IA patterns (e.g. e-commerce, docs site, SaaS)
